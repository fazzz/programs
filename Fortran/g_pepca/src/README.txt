###########
# g_pepca #
###########

```
Pepca based on Koyama et. al. PRE(2008)
```

###Installation
``
make & make install
``
###Requirement
``
LAPACK
``

###How to use
``
g_pepca ene.txt out.txt eigv.txt
ene.txt : energy trajectory
out.txt : pc1, pc2 trajectory
eigv.txt : pc1, pc2 eigen vectors
``

###Parametrs
``
nene : # of energy component
``

###Input format
``
(F12.4,1X,F12.4,1X,F12.4,1X,F12.4,1X,F12.4)
``
